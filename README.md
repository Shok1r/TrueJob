# TrueJob
Frontend part of vk mini app TrueJob

# Установка зависимостей
npm start

# Запуск локально
1. Установить vk tunnel: npm install @vkontakte/vk-tunnel -g 
2. В консоли ввести vk-tunnel --insecure=1 --http-protocol=https --ws-protocol=wss --host=localhost --port=10888 
3. Появившуюся ссылку скопировать и вставить в настройках приложения vk.com/dev
